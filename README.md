
## 📋 Descripción
Sistema de gestión de inventario desarrollado con Laravel. Permite administrar productos y categorías, con control de stock, gráficas interactivas y exportación a CSV.

## 🛠️ Tecnologías
- Laravel 11
- Bootstrap 5
- MySQL
- Chart.js
- Font Awesome
- Git

## 🚀 Instalación en local
1. Clonar el repositorio: `git clone https://github.com/aimee1962/mi_crud.git`
2. Entrar en la carpeta: `cd mi_crud`
3. Instalar dependencias: `composer install`
4. Copiar el `.env.example` a `.env` y configurar la base de datos.
5. Generar clave: `php artisan key:generate`
6. Migrar: `php artisan migrate`
7. Ejecutar: `php artisan serve`
